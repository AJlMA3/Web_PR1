## Описание
Персональный сайт-портфолио, разработанный в рамках практической работы №1. Проект представляет собой современный одностраничный сайт (SPA) с адаптивной версткой, анимациями и автоматическим развертыванием.

**Основные возможности:**
* Семантическая верстка для лучшего SEO и доступности.
* Анимированная Hero-секция с эффектом печатающегося текста.
* Интерактивная временная шкала (Timeline) в разделе образования.
* Визуализация навыков через Progress Bars.
* Карточки проектов с Hover-эффектами.

## Технологии
* **HTML5**: Семантическая структура.
* **CSS3**: Flexbox, Grid, Custom Variables, Animations.
* **JavaScript (ES6+)**: Интерактивность и валидация.
* **GitHub Actions**: Автоматизированный CI/CD процесс деплоя.
