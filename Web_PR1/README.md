## Описание
Персональный сайт-портфолио, разработанный в рамках практической работы №1. Проект представляет собой современный одностраничный сайт (SPA) с адаптивной версткой, анимациями и автоматическим развертыванием.

**Основные возможности:**
* [cite_start]Семантическая верстка для лучшего SEO и доступности[cite: 7, 110].
* [cite_start]Анимированная Hero-секция с эффектом печатающегося текста[cite: 24, 27].
* [cite_start]Интерактивная временная шкала (Timeline) в разделе образования[cite: 73].
* [cite_start]Визуализация навыков через Progress Bars[cite: 64].
* [cite_start]Карточки проектов с Hover-эффектами[cite: 96].
